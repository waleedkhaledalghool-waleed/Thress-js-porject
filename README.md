# 3D Portfolio – Retro TV Edition

A unique interactive **3D portfolio** built with Three.js, designed as a retro television.
<br>
link:https://waleeds3dportfolio.vercel.app/
## How to Interact

- Click the **power button** (bottom right of the TV) to turn it on.
- Click the **channel dial** to switch between channels.
- One of the channels displays my **2D Portfolio**.
- Use the **volume dial** (bottom right) to adjust the sound.
